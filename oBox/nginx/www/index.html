oBox main configuration page.

This is main configuration page. To configure installed components, click on the links below.

Manage User Authentication and Authorization

Use Keycloak which provides interface to manage user authentication and authorization.

Connect to Industrial Data Sources

Connect to PLCs and other industrial data sources using Takebishi Device Gateway and make them accessible via OPC UA Server interface.

Model your Unified Namespace

Use ProsysOPC UA Forge to build your Unified Namespace.

  • Connect to OPC UA servers
  • Harmonize data using information modeling and normalization
  • Provide access to the resulting address space over interfaces: OPC UA Server, REST, OPC UA Pub/Sub

Store and Forward data to databases

Use ogamma Visual Logger for OPC to collect data from OPC UA Servers and store and forward it to the database of your choice.

Use InfluxDB as a historian

Use popular time-series database InfluxDB version 2 or 3 to forward data.

Use InfluxDB version 1.8 as a historian

Use popular time-series database InfluxDB version 1.8 to forward data.

Visualize your data

Use Grafana to visualize real-time and historical data in dashboards.
